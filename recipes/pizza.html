Pizza

Making pizza is honestly really easy! All it includes is:
  • Yeast
  • Water
  • Flour
  • Oil
  • Salt
  • Sugar
  • Cornmeal

Steps

  1. First start will the crust, wel'll use a whole grain crust, you can also use Gluten free crust
  2. Then we will add our favorite sauce, for me in this case i'll be using Alfredo sauce, Pesto sauce and Hummus
  3. Make sure to add veggies! I'll be adding shredded carrots and then diced onions
  4. If you would like this step is optional, add some fruit! I will not be using that for this example
  5. My favorite part! Add some cheese! I'll be using American cheese.

And That's it i hope you enjoy your freshly made pizza! Thank you so much for reading!
